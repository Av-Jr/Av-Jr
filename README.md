# 💫 About Me:
I work on understanding and building systems from the ground up. My interests sit at the intersection of computer science fundamentals, machine learning, and system customization.<br><br>I spend a lot of time exploring core problem-solving concepts such as data structures and algorithms, while also experimenting with practical implementations of machine learning and AI systems. My approach is strongly curiosity-driven: I like taking tools apart, understanding how they work internally, and then bending them to do something slightly unexpected.<br><br>Beyond traditional programming, I experiment with device-level customization and automation. This includes building custom interfaces, lock screens, and system behaviors using tools like KWGT and KLCK, exploring how software interacts with hardware and user interaction layers.<br><br>My projects usually revolve around:<br>• Implementing core CS concepts and algorithms<br>• Exploring machine learning and AI fundamentals<br>• Building small experimental systems and automations<br>• Customizing mobile interfaces and system behavior<br><br>I’m interested in the deeper mechanics of technology — not just using software, but understanding the logic that makes it work.<br>
